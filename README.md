# 华丽大赛！ (Glamour Contest Demo)

一个 **华丽大赛** 网页游戏 Demo，采用 **Low Poly 3D** 美术风格。

## 技术栈

- **React + TypeScript** — UI 框架
- **Three.js** — 3D 渲染（宠物模型、舞台场景）
- **Zustand** — 状态管理
- **Vite** — 构建工具

## 快速启动

```bash
npm install
npm run dev
```

## 游戏流程

```
大厅页面 → 选择比赛 → 选择宠物 → 喂食环节 → 赛前确认 → 表演阶段 → 结算页面
```

## 功能特点

- 🌸 三种比赛类型（风华绝伦赛 / 狂野冲击赛 / 奇思妙想赛）
- 🐕🐹🦝 三只 Low Poly 3D 宠物（雪团团 / 圆咕噜 / 影丸）
- 🍰 五种食物影响宠物状态和心情
- ⚡ 暴击系统（心情越好，暴击率越高）
- 🏆 S/A/B/C/D 五档评级
- 💾 localStorage 记录历史最高分
